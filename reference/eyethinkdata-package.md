# eyethinkdata: Tools for processing behavioural data

A Toolbox for behavioural data. Processes raw downloads from gorilla,
preps behavioural dats for analysis and extracts and scores information
from questionnaires. Performs basic plotting of RDIs (pirate plots), and
correlations
