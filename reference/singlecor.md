# single correlation with r, p value and optionally a bayes factor

single correlation with r, p value and optionally a bayes factor

## Usage

``` r
singlecor(data, x, y, bayes_factor = F)
```

## Arguments

- data:

  data frame with two cols to be correlated

- x:

  name of x column

- y:

  name of y column

- bayes_factor:

  do you want bayes factor calculated, needs BayesFactor package
