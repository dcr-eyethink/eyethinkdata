# takes all of the datatables passed to it, and merges on pid or whetever specified in link, getting rid of duplicates

takes all of the datatables passed to it, and merges on pid or whetever
specified in link, getting rid of duplicates

## Usage

``` r
pid_merge(..., link = "pid")
```

## Arguments

- ...:

  all the data tables

- link:

  cols to link on
